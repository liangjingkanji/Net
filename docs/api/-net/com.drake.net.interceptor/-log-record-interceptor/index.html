网络日志记录器 可以参考此拦截器为项目中其他网络请求库配置. 本拦截器属于标准的OkHttp拦截器适用于所有OkHttp拦截器内核的网络请求库 如果项目中的日志需要特殊情况, 例如请求体加密, 响应体解密, 请继承本拦截器进行自定义 使用Request/Response的peekString函数可以复制请求和响应字符串

在正式环境下请禁用此日志记录器. 因为他会消耗少量网络速度
