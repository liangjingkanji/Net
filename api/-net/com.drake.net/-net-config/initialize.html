Content copied to clipboard

初始化框架 不初始化也可以使用, 但是App使用多进程情况下要求为NetConfig.host或者context赋值, 否则会导致无法正常吐司或其他意外问题

Parameters

host

请求url的主机名, 该参数会在每次请求时自动和请求路径进行拼接(如果路径包含https/http则不会拼接)

context

如果应用存在多进程请指定此参数初始化NetConfig.app

config

进行配置网络请求
